Código Fonte de Interpolação pelo método de La Grange
O programa consiste em receber uma entrada de cordenadas da equação a qual deseja-se interpolar.
A entrada é na forma de pares de dados sendo o primeiro o valor de X e o segundo f(X).
Durante a entrada verifica-se os valores digitados para os mesmos manteram rol crescente para os valores de X
Após a entrada dos valores é exibida uma tabela com os valores de: Número da Entrada; X da entrada; f(X) da entrada e o valor do intervalo de valores possíveis para X.
Após a tabela é requisitado ao usuário um valor de X para fazer a interpolação e obter-se Lk(X).
Essa entrada é verificada no intervalo de I, sendo que para conferência de acuidade os valores de X da tabela podem ser inseridos.
Exibe-se então o valor de Lk(X).É possível ainda entrar com outro valor de X para outras interpolações.
O tamanho para a tabela foi limitado para no máximo 128 pontos de X e f(x).
O programa está com localização de caracteres para Português, portanto números com casas decimais devem ser expressos com ',' e não com '.'.
